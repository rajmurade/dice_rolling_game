# 🎲 Dice Rolling Game

A simple command-line Python game that simulates rolling two six-sided dice.

## Features

- Roll two random dice
- Generates numbers between **1 and 6**
- Play as many times as you want
- Simple and beginner-friendly code

## Requirements

- Python 3.x

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/rajmurade/dice_rolling_game.git
```

2. Navigate to the project folder:

```bash
cd dice_rolling_game
```

3. Run the program:

```bash
python dice_rolling_game.py
```

## Example Output

```

You rolled a 4 and a 2

Do you want to roll the dice? (y/n): y
```
